# Installation
## Linux:

```sh

$ git clone https://github.com/StrinTH/ScrapChat
$ cd ScrapChat
$ pip install -r requirements.txt
#NOTE: xterm should be preinstalled or you can run chat and listen module separately in terminal
$ chmod +x app.sh
$ bash app.sh
```
## Windows:

```sh
#in cmd
> git clone https://github.com/StrinTH/ScrapChat
> cd ScrapChat
> pip install -r requirements.txt
> python3 listen.py
#open another cmd window and
> python3 chat.py

```
